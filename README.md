## 🚀 **Como usar**

### **Instalação**
```bash
# Clonar o repositório
git clone [url-do-repositorio]

# Entrar na pasta
cd presente_site

# Abrir no navegador
# Simplesmente abra index.html em qualquer navegador moderno
```

### **Desenvolvimento**
```bash
# Instalar dependências (opcional)
npm install

# Build de desenvolvimento
npm run build

# Build de produção
npm run build:prod
```
